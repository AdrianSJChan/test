HTML/CSS

HTML stands for Hyper Text Markup Language and its the standard language for creating web pages. It describes the structure and content of the page. Moreover, CSS which stands for Cascading Style Sheets is reponsible for the desgin and 'look' of the webpage. it is able to change settings such as font type and colour of page.
